# SmartRetail AI

An AI-powered retail analytics and recommendation system built using Machine Learning and Streamlit.

Live app link -

## Generate Recommendation Models

Run:

```bash
python src/recommendation/association_rules.py
python src/recommendation/collaborative_filtering.py
```

- Customer Segmentation using RFM + KMeans
- Product Recommendation System
- Collaborative Filtering
- Executive Business Dashboard
- Customer Intelligence AI
- Revenue Analytics
- PCA Visualization
